## Database Schema

-   Visit this -

-   Clone the repository
-   Run **_cd Inventory_**
-   Copy **.env.example** file to **.env** and edit **Database** credentials there
-   Run **_composer install_**
-   Run **_php artisan key:generate_**
-   Run **_php artisan migrate_**
-   Run **_npm install_**
-   Run **_php artisan serve_**
